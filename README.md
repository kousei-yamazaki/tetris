# テトリス（簡易版）

Python と Pygame で動作するシンプルなテトリスゲームです。

---

## 動作環境

- Python 3.x
- Windows / Mac

---

## インストール

### 1. リポジトリをクローン

```bash
git clone git@github.com:kousei-yamazaki/TETRIS.git
cd TETRIS
```

### 2. 依存ライブラリをインストール

```bash
pip install -r requirements.txt
```

---

## 起動方法

```bash
python main.py
```

---

## 遊び方

ゲームを起動するとテトリスフィールドが表示され、ブロックが上から落下してきます。

### 操作キー

| キー | 動作 |
|------|------|
| ← （左矢印） | ブロックを左に移動 |
| → （右矢印） | ブロックを右に移動 |
| ↓ （下矢印） | ブロックを下に移動（加速） |
| ↑ （上矢印） | ブロックを回転 |

### ルール

- ブロックは 0.5 秒ごとに 1 マス自動落下します。
- 横一列がすべて埋まると、その行が消えて上の行が下にシフトします。
- 複数行を同時に消すことができます。
- 新しいブロックが出現した際に既存のブロックと重なるとゲームオーバーです。

### ゲームオーバー

「GAME OVER」と表示されます。ウィンドウを閉じてゲームを終了してください。

---

## ブロックの種類

| 種類 | 形状 | 色 |
|------|------|----|
| I型 | 横一列に4マス | シアン |
| O型 | 2×2の正方形 | 黄色 |
| T型 | T字型 | 紫 |
